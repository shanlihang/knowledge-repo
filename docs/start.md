# 开始使用

#### 🧠 为什么创建这个知识库？

本知识库旨在构建一个 **轻量、自用、高效** 的开发辅助平台。兼容碎片记录与结构化整理，支持 Markdown 写作，Git 管理，长期维护，长期升级。

---

#### 🔧 如何参与 / 自用部署？

欢迎克隆到本地或 Fork 后自己用，或参与内容补充：

```bash
# 克隆仓库到本地
git clone https://github.com/shanlihang/knowledge-repo.git

# 进入项目目录
cd knowledge-repo

# 下载依赖
pnpm install

# 运行项目
pnpm dev

# 打包项目
pnpm build
```
